## 🔢 ***Number Guessing Game in Python***

This is a simple number guessing game built with Python. The program allows the user to choose a range, and then the computer randomly selects a number from within that range. The user must then try to guess the hidden number.

It is a fun beginner-friendly project that helps demonstrate the use of random number generation, user input, conditional statements, and loops.

---

### 🎯 ***Project Description***

The goal of the game is straightforward: guess the number chosen by the computer.

First, the user enters the range they want to play with. The program then picks a random number from that range. After that, the user is prompted to make guesses until the correct number is found.

The game continues by giving feedback after each guess, helping the user know whether their guess is too high or too low.

---

### ⚙️ ***How It Works***

1. The user provides a range for the game.
2. The computer selects a random number from that range.
3. The user enters a guess.
4. The program compares the guess with the chosen number.
5. The program gives feedback to guide the user.
6. The process repeats until the correct number is guessed.

---

### ✨ ***Features***

- User-defined guessing range
- Random number generation
- Interactive user input
- Feedback on each guess
- Loop-based gameplay until the correct answer is found

---

### 🖥️ ***What to Expect When Running the Program***

When the program starts, it will ask the user to enter a range. After that, the game begins immediately.

The user will then be required to guess the number selected by the computer. After each attempt, the program responds based on whether the guess is higher or lower than the target number.

The game ends when the user guesses the correct number.
