# 📊 Math Mini Projects in Python

This folder contains three Python programs that demonstrate fundamental mathematical concepts: binomial coefficients, permutations, and combinations.

Each script is designed to take user input and produce clear results in the terminal.

---

## 1. 🧮 Binomial Coefficients Program (Binomial.py)

This program prompts the user to enter the power of a binomial expression and computes the corresponding binomial coefficients.

### 🔍 What it does
- Accepts a power value from the user
- Calculates the binomial coefficients
- Displays the coefficients in a clear format

### ▶️ What to expect when running it
The program will request a numerical input representing the power of the binomial expression, then output the corresponding coefficients.

### ⚠️ Note
- Printing the full binomial expansion is still in progress  
- Pascal’s Triangle visualization is also under development

---

## 2. 🔢 Permutation Program

This program calculates the permutation of two numbers.

### 🔍 What it does
- Accepts two values (n and r)
- Computes the permutation using the standard formula
- Outputs the result

### ▶️ What to expect when running it
The user will be prompted to input two numbers, and the program will return the number of possible ordered arrangements.

---

## 3. 🧩 Combination Program

This program calculates the combination of two numbers.

### 🔍 What it does
- Accepts two values (n and r)
- Computes the combination using the standard formula
- Outputs the result

### ▶️ What to expect when running it
The user will input two numbers, and the program will return the number of possible selections where order does not matter.
