Here’s what I learned:

🔹 Data Structures

- List → ordered, changeable, allows duplicates
- Tuple → ordered, unchangeable, allows duplicates
- Set → unordered, no duplicates
- Dictionary → key–value pairs for storing related data

I also discovered that these can be used as functions too ("list()", "tuple()", "set()", "dict()"), just like "int", "float", "str", and "bool" 🔄

🔹 Loops

- "while" loop → runs as long as a condition is True 🔁
- "for" loop → iterates over a sequence 📦
- "for-else" → runs the "else" block only if the loop completes without a "break" 🎯

🔹 Control Statements

- "break" → stops the loop immediately 🛑
- "continue" → skips to the next iteration ⏭️
- "pass" → placeholder (does nothing for now) 🧩

🔹 Other Concepts

- Slicing 🔪
- List, string, and dictionary methods 🛠️

One interesting thing I learned:

The ".strip()" method only removes characters from the beginning and end of a string — not the middle ✂️

Another key concept was the "in" keyword 🔍

- With lists/tuples, Python checks items one by one (can be slower as data grows)
- With sets, it checks almost instantly, regardless of size ⚡

While solving the problem sets, I also used:

- ".isupper()" → checks if all characters are uppercase 🔠
- ".isalpha()" → checks if a string contains only letters 🔤
- ".isdigit()" → checks if a string contains only digits 🔢
- "len()" → returns the number of items in an object 📏
