***Here are some of the key things I learned***:

- The "match" operator, which allows comparing a value against multiple patterns 🎯
- The difference between "is" and "==" ⚖️

"is" checks whether two variables point to the exact same object, while "==" checks whether they have the same value 🔍

***Example***: Same value, different objects

a = [6, 4, 3]
b = [6, 4, 3]

print(a is b) # False
print(a == b) # True

***Example***: Same object

a = [5, 1, 8]
b = a

print(a is b) # True
